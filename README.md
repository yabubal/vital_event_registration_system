 Vital Event Registration System (VERS)

A secure and centralized web application designed to digitize the registration of life events, including births, marriages, divorces, and deaths. This system aims to replace manual paper records with a reliable digital database.

 Key Features
Birth Registration:** Issue digital birth certificates and track parentage information.
Marriage & Divorce Tracking:** Securely record legal unions and dissolutions.
Death Registration:** Log mortality data and cause of death for statistical analysis.
Authentication & Authorization:** Secure login for civil registrars and administrators.
Search & Reporting:** Filter events by date, region, or ID to generate vital statistics.

Tech Stack
Backend:** Python (Flask/Django) 
Frontend:** react(vite),(Bootstrap/Tailwind), JavaScript
Database:** MySQL 
Security:** Bcrypt password hashing & Session-based authentication

Project Structure
- `/templates`: Frontend views and forms.
- `/static`: CSS styles and client-side scripts.
- `/models`: Database schemas for event types.
- `/auth`: Logic for user registration and login.
